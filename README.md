# Retail Sales ETL Pipeline

## Project Overview

This project demonstrates a complete ETL (Extract, Transform, Load) pipeline using Python, Pandas, SQLite, and SQL.

## Workflow

CSV
↓

Extract

↓

Transform

↓

Business Analysis

↓

SQLite Database

↓

SQL Queries

↓

Report

## Technologies

- Python
- Pandas
- SQLite
- SQL
- Jupyter Notebook

## Features

- Extract CSV data
- Clean missing values
- Remove duplicates
- Transform data
- Create Total Sales
- Load into SQLite
- SQL Analysis
- Generate Reports
